### Workflow for PA-lower ###

For details on the manual data gather https://rdatools.github.io/tradeoffs/workflow.

( ) Step 1 - Extract the data, shapes, and graph for the state

( ) Step 2 - Set up the state for trade-off analysis
    From 'tradeoffs' run:

    scripts/setup_state PA

( ) Step 3 - Gather additional data points manually

( ) Step 4 - Generate & score an unbiased ensemble
    From 'rdaensemble' run:

    scripts/make_and_score_ensemble.sh --state PA --plan-type lower

( ) Step 5 - Find the trade-off frontiers for the unbiased ensemble
    From 'tradeoffs' run:

    scripts/find_frontiers.sh.sh --state PA --plan-type lower

( ) Step 6 - 'Push' the frontiers of the unbiased ensemble
    From 'rdaensemble' run:

    scripts/push_frontiers.sh --state PA --plan-type lower

( ) Step 7 - Make the analysis artifacts
    From 'tradeoffs' run:

    scripts/make_artifacts.sh --state PA --plan-type lower

( ) Step 8 - Deploy the state
    From 'tradeoffs' run:

    scripts/deploy_state.sh PA

( ) Step 9 - Activate the state in the site
